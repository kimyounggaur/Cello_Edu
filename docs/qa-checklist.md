## 모바일 UI

- [ ] 375px 폭에서 가로 깨짐 없음
- [ ] 하단 탭바가 safe-area를 침범하지 않음
- [ ] 주요 액션이 하단 엄지 영역에 있음
- [ ] 지판 마커 히트영역이 48px 이상임
- [ ] 바텀시트가 화면의 35~45% 정도만 차지함
- [ ] 바텀시트는 핸들/드래그로 닫을 수 있음

- [ ] 색만으로 현/정오답을 구분하지 않음
- [ ] `prefers-reduced-motion`에서 모션이 줄어듦
