# 교사 검수 문서

## 검수 기준

이 앱의 운지 데이터는 `01 Source` 폴더의 표준 운지표와 통합 마스터프롬프트의 `celloData.ts`를 기준으로 합니다.

- 표준 조율: C2, G2, D3, A3
- 인접 현: 완전5도
- 연주자 시점: 왼쪽부터 C, G, D, A
- 닫힌 자세: 손가락 1, 2, 3, 4가 모두 반음 간격
- 1번 손가락 위치: ½=+1, I=+2, II=+3, III=+5, IV=+7
- 앞벌림: 1번 고정, 2·3·4를 반음 위로 이동
- 뒤벌림: 1번만 반음 아래로 이동

## 교사 확인 항목

- [ ] ½~IV 닫힌 자세 표기가 초급 수업 방식과 맞는가
- [ ] 계이름 고정도법 C=도 기본값이 수업 목표와 맞는가
- [ ] “초급 추천 위치”와 “대체 위치”의 구분이 충분한가
- [ ] 오답 피드백 문구가 학생에게 부담 없이 구체적인가
- [ ] 5~7포지션과 엄지 포지션은 퀴즈 정답으로 넣기 전 추가 검수가 필요한가

## 현재 앱에서 일부러 제한한 범위

- 실제 첼로 샘플 대신 Tone.js 합성음 사용
- 5포지션 이상과 엄지 포지션은 설명만 남기고 핵심 퀴즈에는 미포함
- 운지의 문맥 의존성을 고려해 “유일한 정답” 표현을 피함
