# 지판온 · 첼로 운지법 배우기

모바일에서 첼로 운지법과 계이름을 보고, 듣고, 눌러 배우는 PWA입니다. `01 Source` 폴더의 표준 운지표 자료와 `첼로운지법_마스터프롬프트_통합본(오퍼스48).md`의 설계를 기준으로 만들었습니다.

## 핵심 기능

- 모바일 퍼스트 지판 SVG: C·G·D·A, 연주자 시점, 48px 이상 터치 히트영역
- 바텀시트 음 상세: 현, 손가락, 포지션, 음이름, 계이름, 주파수, 재생
- `celloData.ts` 단일 출처: 조율, 주파수, 포지션, 운지, 라벨 계산
- 찾기: 계이름/음이름을 누르면 가능한 지판 위치와 초급 추천 위치 표시
- 연습: A현 1번 B3/시 찾기 기반의 구체적 피드백 예시
- 음계: C/G/D 장조 1옥타브 0·1·3·4 패턴 재생
- 설정/진도: 라벨 체계, ♯/♭, 볼륨, 햅틱, 모션 줄이기, localStorage 저장
- PWA: manifest와 서비스워커 포함

## 실행

```bash
npm install
npm run dev
```

## 검증

```bash
npm test
npm run build
```

`src/lib/celloData.ts`의 `selfTest()`는 개발 모드에서 콘솔에 다음 기준을 PASS/FAIL로 출력합니다.

- C2/G2/D3/A3 = 65.41/98.00/146.83/220.00Hz
- ½ A현 = A#/Bb, B, C, C#/Db
- I G현 = A, A#/Bb, B, C
- IV G현 = D, D#/Eb, E, F
- C/G/D 장조 1옥타브 오름차순

## 자료 기준

- `01 Source/첼로_표준_운지법(Claude).html`: 흑단/황동 디자인 톤, ½~IV 닫힌 자세, 연주자 시점, 완전5도 설명
- `01 Source/첼로_표준_운지법(Chat_GPT).html`: 0~24반음 지도, 1/2~4포지션 표, 앞/뒤 확장 설명, 무프렛 주의
- `01 Source/첼로_표준_운지법(Gemini).html`: 1포지션 중심 초급 표준 운지표

## 주의

앱은 학습용 표준 운지와 초급 추천을 제공합니다. 실제 첼로는 무프렛 악기이므로 정확한 음정은 귀와 선생님의 지도에 따라 미세 조정해야 합니다.
